# NOC App Project

This project aims to create a series of tasks using Clean Architecture with TypeScript and Node.js

# Development
1. Clon and rename .env.template to .env
2. Set your environment variables
3. Run the command ```npm install```
4. Mount the database with
   ```aiignore
   docker-compose up -d 
   ```
5. Run the project with ```npm run dev```
